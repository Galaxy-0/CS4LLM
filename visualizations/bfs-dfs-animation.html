BFS

        
用 queue。先访问离起点最近的一层，再扩散到下一层，所以常用来找无权图最短路径。

      

      

        
DFS

        
用 recursion 或 stack。先沿着一条分支走到底，再回头，常用于树遍历、回溯、检测环。

      

      

        
Visited

        
visited set 防止重复访问。图里如果有环，没有 visited 很容易无限循环。
