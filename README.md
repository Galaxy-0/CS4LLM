# CS4LLM

Use LLM agents to move through a CS / AI industrial-research undergrad map while keeping each course project independently maintainable.

## Workspace Role

This top-level repository is a workspace manager, not a replacement for the child repositories.

- Keep course code and commit history inside each child repository.
- Keep learning notes, roadmap, and workspace utilities at the top level.
- Avoid deleting child `.git` directories or flattening everything into one monorepo.
